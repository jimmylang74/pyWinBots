# 微信 (WeChat) 自动化插件

## 概述
该插件提供对 Windows 微信客户端的自动化控制能力。当前通过 MCP 暴露的能力为向指定联系人发送文本消息。

## 前置条件

1. 使用前需确保微信客户端已启动并登录（扫码登录需人工完成）
2. 如果微信未启动，插件内部会尝试自动启动并连接
3. 如果微信已在运行，插件会自动连接到现有进程

## 可用工具列表

| 工具名 | 功能 | 参数 |
|--------|------|------|
| `weixin_send_message` | 向指定微信联系人发送文本消息 | `contact`: 联系人姓名或昵称, `message`: 消息内容 |

## 使用流程

调用 `weixin_send_message(contact="联系人名称", message="消息内容")` 发送消息。

函数内部会自动完成以下步骤：
1. 聚焦微信主窗口
2. 通过搜索框查找联系人并进入聊天窗口
3. 点击消息输入框，粘贴消息内容
4. 按下回车发送

## 注意事项

- 微信 Windows 客户端使用 CEF (Chromium Embedded Framework) 渲染，部分 UI 元素可能使用自定义控件
- 发送消息时，建议先确认消息内容无误，因为该操作不可撤销
- 确保 Windows 系统的缩放设置不影响 UI 定位（推荐 100%）
